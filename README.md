## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Be sure you have [Postgresql](https://www.postgresql.org/)
- Be sure you have [Ruby](https://www.ruby-lang.org/en/)
- Be sure you have [Rails](https://rubyonrails.org/)
- Be sure you have [Node](https://nodejs.org/en)

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/M-AminAlizadeh/Blog-app.git
  cd Blog-app
```

### Install

Install this project with:


Example command:

```sh
  cd Blog-app
  bundle install
  npm i
```

### Usage

To run the project, execute the following command:

```sh
  rails db:create
  rails db:migrate
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  rspec
  rubocop
```

For more info checkout `.github` folder
